# Gallery Images

This directory is for storing gallery images.

## Image Structure

The gallery expects images to be named according to the category:

- `reinforced-concrete-{1-4}.jpg`
- `block-walls-{1-4}.jpg`
- `sidewalks-&-curbs-{1-4}.jpg`
- `poured-concrete-{1-4}.jpg`
- `masonry-projects-{1-4}.jpg`
- `misc.-commercial-concrete-jobs-{1-4}.jpg`

## Image Recommendations

- **Format**: JPG or PNG
- **Size**: Recommended 1200x800px or similar aspect ratio
- **File Size**: Optimized for web (under 500KB per image)
- **Quality**: High quality construction photos showcasing your work

## Adding Images

1. Add your images to this directory following the naming convention above
2. Images will automatically appear in the gallery on the `/gallery` page
3. For best results, use Next.js Image optimization by importing images in your components

Note: Currently, the gallery uses placeholder gradients. Replace these with actual project photos once available.
